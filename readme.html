Wie das Programm funktioniert

Einlesen

Zuallererst muss die CSV Datei ueber den Button "CSV Datei einlesen" eingelesen werden

Buchung anzeigen

Danach kann die Tabelle angezeigt werden mit der Schaltflaeche "Buchung anzeigen"

Filter

Als naechstes koennen Filteroptionen gewaehlt werden, die die Suche nach bestimmten Eintraegen erleichtern sollen.
Darueber hinaus stehen Optionen zur Verfuegung um bestimmte Werte ersetzen zu koennen. Die Werte werden
dann direkt in die neue Tabelle uebernommen und in die Ausgabe kopiert.

Ausgeben

Der Button erscheint, sobald die Tabelle eingelesen wurde. Dies ermoeglicht eine direkte Ausgabe der Datei im einfachen
Textformat. Dies dient dazu, dass man das Dokument, nachdem Veraenderungen vorgenommen wurden, gleich in einem Format hat,
das man anderweitig einsetzen kann. Bitte beachten Sie, dass JavaScript aus Sicherheitsgruenden keinen direkten Zugriff auf eigene Dateien ermoeglicht.

Kopieren

Kopiert die Ausgabedatei direkt in die Zwischenablage, sodass manuelles Markieren und Kopieren nicht noetig ist.
